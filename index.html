Pozdní sběr suchých nástrojů, které jsou úplně na píču, ale musíme je používat, protože žádné jiné nejsou. Všechny jsou rozbitý, a když to fungují, tak po dvou dnech debugování, hodně skleničkách vína nebo velmi intenzivním trhání vlasů.

Seznam:

  • Vercel – neschopnost platformy pracovat se soukromými balíčky.
  • GitHub Actions – přenastavování permissions jednotlivých workflows je nepoužitelné, debugging neexistuje
  • Gradle – nepotřebuje důvod
  • NPM – autorizace private balíčků z GitHubu se musí provádět přes PAT (Private Access Token)
